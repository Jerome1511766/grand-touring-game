# 🏎️ Grand Touring - Racing Game

A fully-featured 3D racing game built with **Three.js**. Race through dynamic environments, collect checkpoints, and master the track!

## 🎮 Features

- **3D Racing Simulation** - Realistic car physics and handling
- **Dynamic Weather System** - Clear, Rain, and Night modes with visual effects
- **Traffic AI** - Navigate around autonomous traffic cars
- **Checkpoint System** - 10 checkpoints to complete the race
- **Score System** - Earn credits for each checkpoint passed
- **Mini Map** - Real-time track visualization
- **Particle Effects** - Collision and checkpoint celebration effects
- **Responsive Controls** - Keyboard and touch controls
- **Difficulty Scaling** - Boost mode for increased challenge

## 🕹️ Controls

### Keyboard
- **W / ↑** - Gas
- **S / ↓** - Brake
- **A / ←** - Turn Left
- **D / →** - Turn Right

### Mouse/Touch
- Use the on-screen buttons
- Mobile: Touch buttons for accelerate/brake/turn

### Special
- **Boost Speed** - Toggle harder difficulty
- **Reset** - Reset car position and speed

## 🌦️ Weather Effects

| Weather | Effect |
|---------|--------|
| **Clear** | Normal grip and visibility |
| **Rain** | Reduced grip (85%), visual effects |
| **Night** | Low visibility, darker colors |

## 📊 Game Stats

- **Speed Display** - Current velocity in km/h
- **Credits** - 100 points per checkpoint
- **Checkpoints** - Track progress (0-10)
- **Laps** - Completed lap counter

## 🚗 Gameplay Tips

1. **Smooth Turning** - Avoid hard turns at high speed
2. **Weather Awareness** - Adjust driving in rain/night
3. **Collision Avoidance** - Traffic will slow you down
4. **Maintain Speed** - Keep momentum between checkpoints
5. **Use Boost** - Toggle boost mode for extra challenge

## 📦 Installation

1. Clone the repository
2. Open `index.html` in a modern web browser
3. Start racing!

```bash
git clone https://github.com/Jerome1511766/grand-touring-game.git
cd grand-touring-game
# Open index.html in your browser
```

## 🛠️ Technology Stack

- **Three.js** - 3D Graphics Engine
- **WebGL** - GPU Rendering
- **Vanilla JavaScript** - Game Logic
- **HTML5 Canvas** - Mini Map & UI

## 📈 Performance

- Optimized for 60+ FPS on modern hardware
- Shadow mapping for realistic lighting
- Frustum culling for efficient rendering
- Particle pooling for effects management
